About the Project and its Goals

The idea for this project emerged from a challenge faced with sentiment analysis models. While these models provide valuable insights, they often struggle to accurately capture the true sentiment of real-world reviews due to difficulties in interpreting words correctly within various contexts. This can lead to misclassifications, as models may fail to distinguish between positive and negative sentiments depending on how words are used in specific situations.
The goal of this project is to benchmark various sentiment analysis models using real user reviews.
To achieve this, two types of analysis will be employed:
1) Sentiment analysis models, such as Vader and TextBlob,
2) User-based evaluation of review scores.
By comparing these approaches, the project aims to identify which model is best suited for practical use, ultimately providing more accurate and relevant sentiment insights.
*This project is currently in the development phase. At this stage, it focuses on a single scale and a corresponding question of the Net Promoter Score (NPS) type. In the future, additional scales and questions will be integrated into the project, allowing for a broader and more comprehensive evaluation of sentiment analysis models.

Contribution of the Project

This project aims to make a valuable social contribution by providing insights that can guide both individual researchers and businesses in selecting the most suitable sentiment analysis model for their needs. The results of this research will be shared on the project's website, offering transparency and accessibility for those interested in understanding the performance of different sentiment analysis models.
In the future, based on the outcomes of this research, there is the potential to develop a browser extension designed to perform sentiment analysis on product reviews from online marketplaces. This extension would save users time by automatically analyzing thousands of reviews, summarizing key information about the product's advantages and disadvantages, and presenting it in an easily digestible format. This could revolutionize the way users make informed purchasing decisions, enhancing the shopping experience.
